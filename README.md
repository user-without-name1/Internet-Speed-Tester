1. Install Python (if not installed):

• Download from python.org
• During installation, check “Add Python to PATH”

2. Install speedtest-cli library:

powershell
Copy
py -m pip install speedtest-cli
3. Run the Python script:

powershell
Copy
py speed_test.py
• The script will automatically find the best server
• Download and upload speed tests will run with a live loading animation
• Results including download, upload, and ping will be displayed at the end
